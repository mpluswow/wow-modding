AzerothCore Auto Installer Script

This Bash script automates the full setup process for an AzerothCore server on Ubuntu systems.
It's designed to save time and reduce user error by handling everything from package installs to systemd service setup.

Features

When you run this script, it will:

  • Update your system packages
  • Install all dependencies required to build AzerothCore
  • Clone the latest AzerothCore source
  • Prompt you to select and install custom modules
  • Build the core and install binaries
  • Download and extract client data
  • Create a MySQL user acore and assign privileges
  • Install systemd services for easy server management

Install Paths After Completion

  • Binaries: ~/server/azerothcore/env/dist/bin/
  • Configs: ~/server/azerothcore/env/dist/etc/
  • Game Data: ~/server/azerothcore/env/dist/bin/data/

Usage

sudo chmod +x ac-install.sh
sudo ./ac-install.sh

Module Selection

The script supports interactive module selection using dialog. If dialog isn't installed, it falls back to manual input. Available modules include:

  • mod-world-chat
  • mod-eluna
  • mod-ah-bot
  • mod-pocket-portal
  • mod-npc-buffer
  • ...and many more

Post Installation

Once finished, you can manage your servers using:

sudo systemctl start authserver
sudo systemctl start worldserver

To enable them at boot:

sudo systemctl enable authserver
sudo systemctl enable worldserver

This script is ideal for both beginners and experienced server admins looking to get AzerothCore running fast with minimal hassle.
